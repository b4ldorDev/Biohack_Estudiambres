Sobre el Proyecto

Problemática

La artrogriposis múltiple congénita (AMC) es un grupo de trastornos caracterizados por contracturas de las extremidades que afectan principalmente las manos y los pies. Esta condición se asocia con una disminución del movimiento fetal en el útero, lo que ocasiona contracturas articulares que limitan el movimiento.

La AMC generalmente implica debilidad muscular y fibrosis, lo que restringe la movilidad y afecta significativamente la calidad de vida de quienes la padecen. Su principal causa es la acinesia fetal, que lleva al desarrollo de tejido conectivo alrededor de las articulaciones, hecho que restringe el movimiento y aumenta las contracturas articulares.

Nuestra Solución

Desarrollamos una órtesis robótica diseñada para personas con AMC y otras limitaciones en el movimiento del codo o el antebrazo. El dispositivo utiliza sensores de electromiografía (EMG) para detectar las contracciones musculares del usuario y activar un sistema de asistencia motorizada cuando sea necesario.

La órtesis no solo mejora la movilidad, sino que también puede ser utilizada como herramienta de rehabilitación, permitiendo a los usuarios realizar ejercicios terapéuticos con asistencia controlada.

Tecnología

Nuestra órtesis robótica integra tecnologías avanzadas en tres áreas principales:

Componentes Mecánicos

  • Estructura de impresión 3D en PLA reforzado con fibra de carbono
  • Articulaciones de precisión con rodamientos de baja fricción
  • Sistema de ajuste ergonómico adaptable a diferentes anatomías
  • Materiales hipoalergénicos y transpirables en contacto con la piel
  • Diseño modular que permite intercambiar componentes según necesidades

Sistema Electrónico

  • Sensores EMG para detectar señales mioeléctricas
  • Amplificador AD620 para el procesamiento de señales biomédicas
  • Microcontrolador ESP32 para el procesamiento de datos
  • Actuadores eléctricos (servomotores o motores DC)
  • Sensores de posición y movimiento (acelerómetros y giroscopios)
  • Sistema de baterías recargables para uso portátil

Software y Algoritmos

  • Filtros digitales para eliminación de ruido en señales EMG
  • Algoritmos de Machine Learning para reconocimiento de patrones musculares:
    • Árboles de decisión para clasificación de movimientos
    • SVM (Máquinas de Vector Soporte) para identificar activaciones musculares
    • Redes neuronales para modelar relaciones no lineales
    • Random Forest para reducir artefactos por movimiento
  • Interfaz de usuario para configuración de parámetros
  • Sistema de monitoreo y registro de actividad

Marco Regulatorio

El desarrollo de este dispositivo médico se rige por las siguientes normativas:

NOM-241-SSA1-2021

Establece los requisitos de fabricación y controles de calidad por líneas de producción, homologando estándares internacionales como la ISO 13485.

ISO 14971

Establece un proceso sistemático de gestión de riesgos mediante la identificación de peligros potenciales, análisis probabilístico y controles técnicos para mitigación.

ISO 10993-1

Regula la evaluación biológica mediante una matriz de riesgo que considera duración del contacto con el dispositivo y el tipo de exposición.
